AeroACORN

Aerobiology and remote sensing for early forecasting of ACORN production

About the Project

AeroACORN aims to develop the first early warning system for ACORN production using integrated aerobiological monitoring and satellite remote sensing data.

Objectives

  • Develop predictive models for acorn production
  • Integrate pollen monitoring with remote sensing indices
  • Support sustainable forest and agroforestry management

Methodology

The project combines long-term aerobiological datasets, phenological observations, and Earth Observation data to model production variability under climate change scenarios.
